# BemolChannel

An api that registrate the client data in a database


## 🛠 Tecnologies used

* Node.js
* React.js
* SQLite


## Features

 - [x]  Login page
 - [x]  Register page
 - [x]  Update data page
 - [x]  API that get your address from Correios's API
